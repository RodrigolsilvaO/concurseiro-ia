\# ConcurseiroIA



Sistema web para auxiliar concurseiros na organização dos estudos a partir do edital.



\## Objetivo



O projeto tem como objetivo permitir que o usuário cadastre concursos, matérias e tópicos do edital, registre sessões de estudo, acompanhe tempo bruto e líquido, visualize sua evolução e futuramente utilize IA para interpretar editais em PDF e sugerir planos de estudo personalizados.



\## Funcionalidades previstas



\- Cadastro de concursos

\- Cadastro de disciplinas

\- Cadastro de tópicos do edital

\- Controle de tempo bruto e líquido de estudo

\- Registro de pausas

\- Dashboard de evolução

\- Upload de edital em PDF

\- Interpretação de edital com IA

\- Sugestão automática de plano de estudo



\## Tecnologias previstas



\- .NET Web API

\- Angular

\- SQL Server

\- Entity Framework Core

\- Git e GitHub

\- Integração futura com IA
